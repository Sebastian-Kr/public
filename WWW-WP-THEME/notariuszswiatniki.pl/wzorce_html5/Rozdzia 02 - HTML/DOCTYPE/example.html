DOCTYPE

Dla dokumentów XHTML zalecam następujący typ dokumentu:
  • Powoduje sprawdzanie składni dokumentu wg standardu XHTML.
  • Jest to tzw. typ przejściowy, który zezwala na na używanie kilku przydatnych atrybutów, takich jak start w elementach <ol> i value w elementach <li>.
  • Włącza w przeglądarkach tryb prawie zgodny ze standardami, który jest niezbędny, jeśli chce się używać arkuszy stylów.

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

Dla dokumentów HTML zalecamy dwa następujące typy dokumentu „ ”.

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/1999/REC-html401-19991224/loose.dtd">
